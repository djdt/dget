# DGet
Calculate compound deuteration from ToF-MS data.

Uses [molmass](https://github.com/cgohlke/molmass) for isotopic mass calculations.

# Installation
DGet is available on PyPI

```pip install dget```

# Usage

Basic usage:

```dget <formula> <path-to-tof-data>```

To see CLI options:


```dget --help```

DGet!

About

DGet calculates sample deuteration from high-res mass spectrometer data. To use the web application simply use the inputs to enter a formula and upload a data file.

Usage

Formula - molecular formula of the fully deuterated sample, see molmass for more details.

Adduct - molecular species, use 'Auto' to guess the adduct from the base peak of the mass spectra.

Re-align - shifts ToF data to better align with the formula's predicted mass. Please calibrate your instruments instead of using this.

MS Data File - delimited text file with a column of masses (Da) and column of signal/intensity values.

File delimiter - delimiter used in the file, DGet will try to guess on file upload.

Skip rows - number of rows before data, DGet will attempt to guess on file upload.

Mass/signal column - column in file containing the mass/signal data.

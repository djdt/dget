DGet!

About

DGet calculates sample deuteration from high-res mass spectrometer data. To use the web application, enter a formula and upload a mass spectra text file.

Usage

Formula - molecular formula of the fully deuterated sample, see molmass for more details.

Adduct - molecular species, use 'Auto' to guess the adduct from the base peak of the mass spectra.

Upload MS data - DGet expects a delimited text file with at least a column of masses (Da) and column of signal/intensity values.

MS data options - the following options control how data is read from the uploaded file. DGet will attempt to guess options on file upload.

  • Delimiter - text delimiter used in the data file.
  • Skip rows - number of rows in file before data.
  • Mass/Signal column - column in file containing the mass/signal data.

Re-align MS data - shifts MS data to better align with the formula's predicted mass. Please calibrate your instruments instead of using this.

Subtract MS baseline - performs baseline subtraction, calculated as the 25th percentile of the target region.
